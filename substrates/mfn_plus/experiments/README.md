# Experiments

## Quick Start

```bash
# One command — full verification
bash ci.sh

# Or individual steps:
uv run python experiments/reproduce.py     # Canonical reproduction
uv run python experiments/adversarial.py   # Adversarial validation
```

## Canonical Reproduction

`reproduce.py` runs a fixed simulation through all subsystems and verifies output matches `expected_output.json`.

- **Seed**: 42, **Grid**: 32x32, **Steps**: 60
- **Field hash**: `b407b808c7c8a03f` (SHA256, must be exact)
- **Float tolerance**: 0.01

If it fails, something in the pipeline changed. Investigate before merging.

## Adversarial Validation

`adversarial.py` checks 6 invariants across multiple seeds:

1. **Determinism**: same seed = bitwise identical output
2. **NaN/Inf safety**: 50 seeds, no NaN/Inf in any field
3. **Causal gate**: 20 seeds, all outputs have valid labels/scores
4. **Shape invariants**: grid 8-64, field shape always matches spec
5. **Bounded metrics**: 10 seeds through UnifiedEngine, all in range
6. **Perturbation stability**: nearby seeds produce consistent labels

## Outputs

| File | Generated by | Committed |
|------|-------------|-----------|
| `expected_output.json` | `reproduce.py` (first run) | Yes (baseline) |
| `actual_output.json` | `reproduce.py` (every run) | No (.gitignore) |
| `results/adversarial_results.json` | `adversarial.py` | No (.gitignore) |
