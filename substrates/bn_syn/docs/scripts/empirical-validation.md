# scripts/empirical_validation.py

- Invocation: `python -m scripts.empirical_validation --help`
- Purpose: Convert aggregate benchmark scenario records into deterministic validation metrics and a calibration report.
- Inputs:
  - `--input-json`: Benchmark JSON generated by `python -m benchmarks.run_benchmarks --json <path>`.
- Outputs:
  - `--output-json`: Objective metrics JSON (`median_wall_time_sec`, `stability_integrity_index`, `review_load_index`) plus branch-calibration decisions.
  - `--output-md`: Human-readable markdown report for reviewer inspection.

## Example

```bash
python -m benchmarks.run_benchmarks \
  --scenario ci_smoke \
  --repeats 1 \
  --warmup 0 \
  --json results/stress_ci_smoke.json

python -m scripts.empirical_validation \
  --input-json results/stress_ci_smoke.json \
  --output-json proof_bundle/empirical_validation/objective_metrics.json \
  --output-md proof_bundle/empirical_validation/report.md
```

## Determinism

- Rejects malformed rows and missing required fields.
- Rejects negative and non-finite numeric values.
- Writes newline-terminated JSON with sorted keys.
- Produces deterministic markdown for identical inputs.
