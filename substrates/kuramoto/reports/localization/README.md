# Localization Reports Directory

This directory contains **auto-generated** localization coverage reports.

## ⚠️ Important

**DO NOT COMMIT** generated localization reports to git. These files include:
- `coverage.json` - Localization coverage metrics
- Any other auto-generated `*.json` files

These reports are generated by localization scanning tools and CI workflows.

## Purpose

Localization reports track:
- Coverage of UI strings and messages
- Translation completeness per locale
- Missing or outdated translations

## Viewing Results

Check localization reports in:
1. **CI Artifacts** - GitHub Actions workflow artifacts
2. **Local Runs** - Generated after running localization scanning tools

## Configuration

Localization settings and tooling are defined in:
- `docs/localization/` - Localization documentation
- Relevant CI workflows for i18n validation

Generated files are automatically ignored by git (see `.gitignore`).
