# Performance Test Results Directory

This directory contains **auto-generated** performance test results from CI and local test runs.

## ⚠️ Important

**DO NOT COMMIT** generated performance reports to git. These files are:
- `performance_report.json` - Structured test results
- `performance_summary.md` - Human-readable test summary
- Any other `*.json` or `*.csv` files with test data

These files are automatically generated by:
- `.github/workflows/multi-exchange-replay-regression.yml`
- Local performance test runs via pytest

## Viewing Results

Performance results are available through:
1. **GitHub Actions Artifacts** - Check the workflow run's artifacts tab
2. **Local Test Runs** - Generated in this directory after running performance tests
3. **CI Summary** - Displayed in GitHub Actions job summaries

## Configuration

Performance test budgets and scenarios are defined in:
- `tests/perf/` - Performance test implementations
- `bench/` - Benchmark scenarios and configurations
- `docs/PERFORMANCE_REGRESSION_GUIDE.md` - Performance testing guide

## Retention

Generated files in this directory are automatically ignored by git (see `.gitignore`).
Clean up old reports manually if disk space is a concern.
