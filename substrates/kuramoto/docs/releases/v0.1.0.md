# Release v0.1.0 — Initial Public Preview

**TradePulse v0.1.0** marks the first public release of our enterprise-grade algorithmic trading platform with geometric market indicators.

---

## 🎯 What is TradePulse?

TradePulse is a production-ready algorithmic trading framework that uses advanced geometric and information-theoretic indicators to classify market regimes and generate trading signals.

Unlike traditional technical analysis, TradePulse applies concepts from:
- **Kuramoto Synchronization** — Detecting collective oscillation patterns in price dynamics
- **Ricci Flow Geometry** — Measuring market curvature and topological transitions
- **Entropy Analysis** — Quantifying information content and regime changes

### Core Features

- **50+ Indicators** — Geometric (Kuramoto, Ricci) + classical technical indicators

## 🚀 Quick Start

```bash
# Clone and install
git clone https://github.com/neuron7x/TradePulse.git
cd TradePulse
pip install -e .

# Analyze market data
python -m interfaces.cli analyze --csv sample.csv --window 200

# Run a backtest
python -m interfaces.cli backtest --csv data.csv --fee 0.0005
```

### Minimal Python Example

```python
import numpy as np
import pandas as pd
from core.indicators.kuramoto_ricci_composite import TradePulseCompositeEngine

# Create market data
index = pd.date_range("2024-01-01", periods=720, freq="5min")
prices = 100 + np.cumsum(np.random.normal(0, 0.6, 720))
bars = pd.DataFrame({"close": prices, "volume": np.random.lognormal(9.5, 0.35, 720)}, index=index)

# Analyze
engine = TradePulseCompositeEngine()
signal = engine.analyze_market(bars)

print(f"📊 Phase: {signal.phase.value}")
print(f"🎯 Confidence: {signal.confidence:.3f}")
print(f"📈 Entry Signal: {signal.entry_signal:.3f}")
```
