# Enablement Artifacts Index

This folder stores templates, ownership matrices, schedules, and assessments that support the engineering enablement program. Documents are version-controlled and follow the documentation governance workflow.

## Contents

- `ownership_matrix.xlsx` – Maintains accountable leads per role track, mentorship cohort, and ritual.
- `cadence_calendar.md` – Tracks seminar, pairing, and rotation schedules with capacity constraints.
- `assessment_rubrics/` – Level-specific competency rubrics and evaluation forms.
- `templates/` – Standardized forms for Individual Growth Plans, mentorship agreements, and incident lesson logs.

> **Note:** Sensitive personnel data should remain in the HRIS; store only sanitized, shareable artifacts here.
