# CNS Stabilizer Artifacts

## Overview

This dataset contract defines artifacts generated by the Central Nervous System (CNS) Stabilizer component of TradePulse's neuromodulation system. These artifacts capture free energy dynamics, phase transitions, and system stability metrics used for thermodynamic control and monitoring.

## Artifacts

### artifacts/cns_stabilizer/eventlog_sample.json

**Format**: JSON (JavaScript Object Notation)

**Description**: Sample event log entry from the CNS Stabilizer showing a complete stability checkpoint with Genetic Algorithm (GA) phase information, free energy delta measurements, and system integrity metrics.

**Schema**:
```json
{
  "epoch": integer,           // Timestep or iteration number
  "ga_phase": string,         // Genetic algorithm phase: "sample", "optimize", "converge"
  "data": {
    "delta_f": float,         // Free energy change (ΔF)
    "phase": string,          // System phase: "stable", "transition", "critical"
    "action": string,         // Stabilizer action: "pass", "adjust", "halt"
    "integrity": float,       // System integrity score [0.0, 1.0]
    "margin": float,          // Safety margin from threshold
    "threshold_after_pid": float,  // Dynamic threshold after PID control
    "chunk_size": integer,    // Processing chunk size
    "processed_samples": integer,  // Samples processed in this epoch
    "latency": float          // Processing latency in seconds
  },
  "timestamp": float          // Unix timestamp with fractional seconds
}
```

**Characteristics**:
- Single-epoch snapshot of CNS stabilizer state
- Demonstrates stable phase operation
- Shows PID-adjusted threshold dynamics
- Includes performance metrics (latency, chunk size)
- Timestamp in Unix epoch format

**Use Cases**:
- CNS stabilizer unit testing
- Event logging format validation
- Thermodynamic monitoring dashboard examples
- Free energy analysis demonstrations
- System stability metric visualization

**Generation**: Captured from actual CNS stabilizer execution during stable market conditions with GA sampling phase.

### artifacts/cns_stabilizer/delta_f_heatmap.csv

**Format**: CSV (Comma-Separated Values)

**Description**: Free energy delta (ΔF) heatmap data mapping epochs to free energy changes, phase states, and margin metrics. Used for visualizing system stability over time. This file is the single canonical source for the heatmap data (consolidated from the duplicate sample file to keep one dataset).

**Schema**:
- `epoch` (integer): Timestep or iteration number
- `delta_f` (float): Free energy change measurement
- `phase` (string): System phase classification
- `margin` (float): Distance from critical threshold

**Characteristics**:
- 1 data point (sample dataset)
- Stable phase classification
- Zero margin indicating nominal operation
- Suitable for heatmap visualization

**Use Cases**:
- Thermodynamic stability visualization
- Phase transition detection
- Free energy landscape mapping
- System health monitoring
- Time-series analysis of stability metrics

**Generation**: Extracted from CNS stabilizer telemetry during normal operation.

## Thermodynamic Context

The CNS Stabilizer is part of TradePulse's Thermodynamic Autonomic Control Layer (TACL) which ensures system stability through free energy minimization principles. Key concepts:

- **Free Energy (F)**: Variational free energy from active inference framework
- **ΔF (Delta F)**: Change in free energy between timesteps; must be non-positive for stability
- **Phase States**: System operational states (stable, transition, critical)
- **PID Control**: Proportional-Integral-Derivative controller adjusts thresholds dynamically
- **Genetic Algorithm Integration**: Evolutionary optimization in stabilizer feedback loop

## Integration Points

```python
# Example: Loading CNS stabilizer event log
import json
from pathlib import Path

event_log_path = Path("artifacts/cns_stabilizer/eventlog_sample.json")
with event_log_path.open() as f:
    event = json.load(f)
    
print(f"Epoch {event['epoch']}: ΔF = {event['data']['delta_f']:.4f}")
print(f"Phase: {event['data']['phase']}, Integrity: {event['data']['integrity']:.2f}")
```
