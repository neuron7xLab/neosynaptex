# API Changelog

## 2025.1 — 2025-02-01
Added asynchronous prediction workflow and raised signal rate limits.

Compatibility: `stable`

### Highlights
- Introduced POST /v1/predictions with webhook callbacks.
- Increased GET /v1/signals cache window and burst limits.

## 2024.4 — 2024-11-15
Initial stable release of the TradePulse public API.

Compatibility: `stable`

### Highlights
- Released GET /v1/signals/{symbol} endpoint.
